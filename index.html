Aplikasi Telah Diubah Menjadi Bot Command-Line (CLI)

Aplikasi ini sekarang berjalan di terminal (command line), bukan di browser web. Antarmuka web telah dinonaktifkan.

File utama yang baru adalah server.js.

Cara Menjalankan Bot:

  1. Buat file bernama .env di direktori proyek.
  2. Di dalam file .env, tambahkan baris berikut: API_KEY=KUNCI_API_GEMINI_ANDA (ganti dengan kunci API Anda yang sebenarnya).
  3. Buka terminal atau command prompt di direktori proyek Anda.
  4. Jalankan perintah npm install untuk menginstal semua dependensi yang diperlukan.
  5. Setelah instalasi selesai, jalankan npm start untuk memulai bot.
  6. Ikuti petunjuk di terminal untuk berinteraksi dengan bot.
